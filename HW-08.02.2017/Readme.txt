Двусвязный список:
Методы push_back, push_front, pop_back, pop_front, front, back работают за O(1),
insert, erase, find, clear - O(n), где n - размер списка.
Методы pop_front, pop_back, erase, insert возвращают 1 в случае успешного выполнения операции, 0 при ошибке.
front, back и оператор [] в случае ошибки завершают работу программы.
------------------------------------------------------------------------------------------------------------
Очередь - двусвязный список, оставлены только необходимые (push_back, pop_front, front) методы.
------------------------------------------------------------------------------------------------------------
Односвязный список - в целом похож на двусвязный список, однако операция pop_back работает за O(n).
------------------------------------------------------------------------------------------------------------
Cтек - односвязный список с необходимыми (push_back, pop_back, back) методами.
